## Cấu trúc thư mục

```
Bước 1: Tải folder về máy
Bước 2: chạy lệnh npm install để tải tất cả packages cần thiết vào folder
Bước 3: chạy lệnh node start để chạy dự án
```

```
|--- node_modules/
|--- src/
│   |--- config/
│   |--- controllers/
│   |--- middlewares/
│   |--- models/
│   |--- routes/
│   |--- services/
|--- .env
|--- nodemon.json
|--- app.js
|--- package.json
|--- package-lock.json
|--- README.md
```
